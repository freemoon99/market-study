# 당일 시장 정리

유목민님식 당일 시장 복기를 빠르게 하기 위한 주식 공부용 웹앱입니다.  
당일 한국 증시에서 상한가 종목과 거래량 1000만주 이상 종목을 자동으로 모으고, 종목별 재료와 학습 노트를 한 화면에서 정리할 수 있게 만든 프로그램입니다.

## 목적

매일 반복되는 시장 정리의 전처리를 자동화해서 사용자가 더 중요한 공부에 집중하도록 돕습니다.

- 오늘 어떤 종목이 강했는지 빠르게 확인
- 상한가와 거래량 급증 종목의 원인 후보 수집
- 종목별 차트, 수급, 기본 지표, 회사 개요 확인
- 당일 테마와 시장 분위기 파악
- 직접 작성한 의견과 복기 기록을 날짜별로 누적

## 주요 기능

- 네이버 증권 상한가/거래상위 페이지 기준 종목 수집
- 상한가, 거래량 1000만주 이상, 총 종목 수 요약
- 당일 테마 자동 추출 및 테마 태그 필터
- 종목별 현재가, 등락률, 거래량, 전일/평균 대비 거래량 변화 표시
- 캔들/거래량 확인용 인터랙티브 차트
- 당일 뉴스 기반 상승/거래량 원인 후보 자동 등록
- 원인 후보 수정, 삭제, 직접 추가
- 시가총액, PER, PBR, ROE, EPS 등 기본 분석 지표와 도움말
- 네이버 금융 기업개요 기반 회사 소개, 사업군, 실적 포인트 자동 작성
- 사용자가 직접 작성하는 `내 의견` 중심 학습 노트
- 날짜별 기록 저장 및 이전 기록 조회
- `n일째 상한가`, `n일째 거래량 1000만주` 같은 연속 출현 태그
- 가격 모멘텀과 거래량 에너지 기준 주식 4분면
- 상대강도와 모멘텀 기준 RRG 그래프
- 다크모드/라이트모드
- Markdown 요약 파일 브라우저 즉시 다운로드

## 데이터 저장

로컬 실행에서는 SQLite 파일에 저장하고, Render 배포 환경에서는 `DATABASE_URL`이 설정된 경우 Postgres에 저장합니다.

Markdown 내보내기는 서버에 파일을 남기지 않고, 버튼을 누른 시점에 브라우저에서 바로 다운로드됩니다.

## 데이터 출처

- 네이버 증권 상한가 페이지
- 네이버 증권 거래상위 페이지
- 네이버 금융 종목 차트/기업개요
- Google News RSS

수집 대상은 공부 후보에 집중하기 위해 ETF, ETN, 스팩류를 제외합니다. 외부 페이지 구조나 네트워크 상태에 따라 수집 결과가 달라질 수 있습니다.

## Render 배포 메모

Render 무료 Postgres를 사용할 수 있지만, 무료 DB는 1GB 제한과 30일 만료 제약이 있습니다. 장기 기록을 보관하려면 유료 DB 또는 별도 백업 전략이 필요합니다.
